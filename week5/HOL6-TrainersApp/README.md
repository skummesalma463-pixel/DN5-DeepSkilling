# HOL 6 - Trainers App (React Router)

## Objectives
- Implement a simple navigation menu
- Add basic routes with react-router-dom
- Pass parameters via URL with useParams

## Routes
- / -> Home
- /trainers -> TrainersList (clickable names)
- /trainers/:id -> TrainerDetails (looked up from mock data via useParams)

## How to run
npm install
npm start
