# HOL 9 - Cricket App (ES6 Features)

## Objectives
- Use map() and arrow functions
- Apply array destructuring
- Merge arrays with the spread operator

## Task
- ListofPlayers: renders 11 players via map()
- Scorebelow70: filters players with score <= 70 via arrow function
- OddPlayers/EvenPlayers: destructure the IndianTeam array to pick alternating players
- IndianPlayers: merges T20Players and RanjiTrophyPlayers via spread (...)
- Toggle between the two views by flipping `flag` in App.js

## How to run
npm install
npm start
