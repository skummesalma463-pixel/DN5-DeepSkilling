# HOL 5 - Cohort Dashboard (CSS Modules)

## Objectives
- Style react components using CSS Modules
- Apply styles via className and inline style props

## Task
Build a CSS Module (CohortDetails.module.css) with a .box class (300px width,
inline-block, 10px margin, 10/20px padding, 1px black border, 10px radius) and
a dt tag selector (font-weight 500). Cohort heading is green when status is
"ongoing", blue otherwise.

Note: this lab in the original doc used a pre-built starter zip attached as an
.emf image, which could not be extracted from the Word document. This
implementation reconstructs the same requirements from scratch as a working
project.

## How to run
npm install
npm start
