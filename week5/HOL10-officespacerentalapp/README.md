# HOL 10 - Office Space Rental App (JSX)

## Objectives
- Use JSX syntax to create React elements and attributes
- Render a JS object's data inside JSX
- Apply inline/conditional CSS classes in JSX (red rent if <= 60000, green otherwise)

## Note
Add any office image as src/office.jpg (the original doc referenced a local
image not recoverable from the Word file) — the component will render it at
25% width/height above the details.

## How to run
npm install
npm start
