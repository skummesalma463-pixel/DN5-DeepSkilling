Lab 5: Retrieving Data from the Database
============================================

Steps:
1. Open Program.cs in your RetailInventory project.
2. Replace its contents with Program.cs from this folder (this replaces
   the Lab 4 insert code with retrieval code).
3. Run the app:
       dotnet run
4. Expected console output:
       All products:
       Laptop - Rs.75000
       Rice Bag - Rs.1200

       Found by Id=1: Laptop
       First product over Rs.50000: Laptop

Screenshot the terminal output.

Note - keeping Lab 4's insert data available:
Since Lab 4's Program.cs and Lab 5's Program.cs both replace the same
file, if your instructor wants to see both running, run Lab 4's
Program.cs first, screenshot, then swap in Lab 5's Program.cs and run
again (matches how the labs are presented as sequential steps).
